# MetaCommunity 
这是一个元社区的模板，实现了多人在线交互，组件自定义布局，用户自主移动控制，NPC点击交互功能\
#前端文件:根目录下文件(除server) \
npm i \
npm run dev\
#后端文件:sever文件夹\
npm i\
node app
